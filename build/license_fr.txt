3. DESCRIPTION DU LOGICIEL

3.1 Fonctionnalité de l'installateur
Le Logiciel est un programme d'installation conçu pour faciliter l'installation de l'application Local Operator sur votre/vos appareil(s). L'installateur peut effectuer les fonctions suivantes :

a) Vérifier les exigences système et la compatibilité ;

b) Télécharger les composants nécessaires pour l'application Local Operator ;

c) Configurer votre système pour des performances optimales de l'application Local Operator ;

d) Créer les répertoires et fichiers nécessaires sur votre appareil ;

e) Enregistrer l'application Local Operator auprès de votre système d'exploitation ;

f) Installer les dépendances requises pour le bon fonctionnement de l'application Local Operator.

3.2 Collecte de données par l'installateur
Pendant le processus d'installation, le Logiciel peut collecter certaines informations sur votre appareil, telles que :

a) Type et version du système d'exploitation ;

b) Spécifications matérielles ;

c) Configuration réseau ;

d) Logiciels existants susceptibles d'interagir avec l'application Local Operator ;

e) Informations sur le succès ou l'échec de l'installation.

Ces informations sont utilisées uniquement dans le but de faciliter le processus d'installation et d'améliorer le Logiciel. Pour plus d'informations sur les pratiques de collecte de données, veuillez consulter notre Politique de confidentialité.
